Challenge 5: Accessibility Concerns

Most of the elements in HTML have accessibility built into it like img. Some of the text elements like for the chess pieces or some links are technically human readable, but not a language, so it would be hard for a machine to read to a person. A screen reader might have issues determining what order to read the screen in for some webpage layouts that isn't necessarily meant to be read left-right, top-bottom.
